# Coffee Report CLI

CLI приложение для анализа метрик видео на YouTube. Формирует отчет о кликбейтных видео (CTR > 15% и удержание < 40%).

# Запуск
python cli.py --files stats1.csv stats2.csv --report clickbait

# Пример вывода

Программа выводит таблицу в консоль. Видно 4 колонки: заголовок, CTR и удержание.

Пример данных в отчете:
- Секрет который скрывают тимлиды: CTR 25.0%, удержание 22%
- Как я спал по 4 часа: CTR 22.5%, удержание 28%
- Купил джуну макбук и он уволился: CTR 19.0%, удержание 38%
- Я бросил IT и стал фермером: CTR 18.2%, удержание 35%
