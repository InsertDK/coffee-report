# Coffee Report CLI

CLI приложение для анализа метрик видео на YouTube. Формирует отчет о кликбейтных видео (CTR > 15% и удержание < 40%).

# Запуск
python cli.py --files stats1.csv stats2.csv --report clickbait

# Пример вывода
+--------------------------------------------------------+-------+-----------------+
| title | ctr | retention_rate |
+========================================================+=======+=================+
| Секрет который скрывают тимлиды | 25.0 | 22 |
+--------------------------------------------------------+-------+-----------------+
| Как я спал по 4 часа и ничего не понял | 22.5 | 28 |
+--------------------------------------------------------+-------+-----------------+
| Купил джуну макбук и он уволился | 19.0 | 38 |
+--------------------------------------------------------+-------+-----------------+
| Я бросил IT и стал фермером | 18.2 | 35 |
+--------------------------------------------------------+-------+-----------------+
